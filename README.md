# Skkeleton

skkeleton is a Vim/Neovim plugin that provides Japanese input environment called
SKK.

It's heavily inspired from [ddskk](https://github.com/skk-dev/ddskk) and
[eskk.vim](https://github.com/vim-skk/eskk.vim).
